计算当前dag的最大并发度

Parameters
elements
Returns

计算思路如下：

  1. 列出来所有的可行路径(paths)
  2. 根据路径，生成一张全连通图，然后将图取反，得到对应的补图(reGraph)
  3. 计算补图的最大团中元素个数(maxCliqueSize)，即为当前dag的最大并行度
