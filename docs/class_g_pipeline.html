◆ run()

CStatus GPipeline::run ( )
overridevirtual

流程处理函数

  1. 将所有 GElement 的状态设定为 NORMAL
  2. 将所有的 GParam 设置为初始值
  3. 执行dag逻辑
  4. 将所有 GElement 的状态恢复为 NORMAL
  5. 等到所有异步事件执行结束
  6. 将所有的 GParam 复原

Implements CObject.
